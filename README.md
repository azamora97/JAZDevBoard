# Proyecto JAZ-DevBoard

Proyecto curso de React Avanzado

1. Clonar el repositorio:
   git clone https://github.com/azamora97/JAZDevBoard
2. Ingresar a la carpeta del proyecto:
   cd jaz-devboard
3. Instalar los modulos de node ejecutando el siguiente comando:
   npm install
4. Correr el proyecto:
   npm run dev
5. Abrir en el navegador la URL que se muestra en la consola:
   Se mostrar una URL similar a la sguiente: http://localhost:5173/
